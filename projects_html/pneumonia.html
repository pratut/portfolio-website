Pneumonia Detection (CNN)

ResNet-18 based chest X-ray classifier achieving 92% accuracy with optimized preprocessing.

Project Details

This project uses convolutional neural networks to classify chest X-rays as pneumonia or normal. The preprocessing pipeline was optimized to reduce training time by 10% while maintaining 92% accuracy.

Technologies Used:

  • Python, PyTorch
  • ResNet-18
  • Image preprocessing and augmentation
  • Model checkpointing and evaluation metrics
